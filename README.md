# CPU-GPU Compression Throughput Benchmark

## Overview
This project is part of a Master's thesis on evaluating the impact of compression and decompression on CPU–GPU data transfer throughput.

The goal is to analyze how different data transfer strategies and compression techniques affect performance when moving data between CPU and GPU.

---

## Project Structure

- `src/`
  - Contains the core benchmark implementations:
    - `baseline_benchmark.cu`
    - `averaged_benchmark.cu`

- `results/`
  - Stores output results generated by the benchmarks
  - Example:
    - `avg_results.csv`

- `docs/` (optional)
  - Notes or documentation related to the project

---

## Benchmarks Included

### 1. Baseline Benchmark
- Measures raw CPU–GPU transfer performance
- Includes:
  - CPU memory copy
  - Host-to-Device (H2D) transfer
  - Device-to-Host (D2H) transfer
  - GPU memory throughput

### 2. Averaged Benchmark
- Executes multiple iterations for stable performance measurements
- Produces averaged results
- Outputs results in structured format (e.g., CSV)
